# Learn React useState

**Live Form Preview – See your state updates in action**

---

## Project Overview

This is an **interactive React project** designed to help beginners understand how **`useState`** works in React.  
The app simulates a **live form editor**, showing the difference between **Draft State** and **Applied State**, while also visualizing changes in a **Virtual DOM Tree**.

It’s a **fun and educational project** to see how React updates the UI efficiently and how state changes flow through components.

---

## Key Features

- **Draft vs Applied State**: Edit a profile in draft mode and apply changes to see the difference.
- **Interactive Virtual DOM Tree**: Highlights which part of the UI actually changes after a render.
- **Real-time Render Counter**: Shows how many times the UI re-rendered.
- **Professional UI**: Clean design with subtle animations and visual cues.
- **Responsive Layout**: Works on desktop, tablet, and mobile screens.

---

## Technologies Used

- **React** (Functional Components & Hooks)
- **useState** for state management
- **CSS Grid & Flexbox** for layout
- **CSS Animations** for interactive highlights
- **Responsive Design** principles
